G&B Boutique

Description

Application web qui permet à un internaute de s’inscrire et de commander dans une E-Boutique. Celui-ci pourra :
-s'inscrire sur le site web (juste login, mot de passe au minimum)
- afficher la liste des produits
- remplir un panier d’achat à partir d’articles sélectionnés
- visualiser le contenu de ce panier d’achat,
- envoyer sa commande par mail et sera enregistrée dans une base de données.

Technologies : JAVA :EE, HTML, CSS, MySql, Tomcat Server ,Git, Netbeans.
Responsabilité  : Modélisation de la Base de donnée, Gestion du panier d’achat et la commande.
