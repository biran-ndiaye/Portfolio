File d'attente

Description

Application Mobile à deux couches : employé et client, permettant de gérer la file d’attente des clients d’un garagiste automobile qui changent leur pneu de voiture. Le client pourra s’inscrire ou se connecter, choisir la voiture correspondante à la sienne et enfin s’il veut changer son pneu , entrer dans la file d’attente et il verra tous les inforamtions(temps d’attente et position occupé) .Quant à l’employé du garagiste, des qu’il est disponible, le signaler dans l’application pour permettre l’avancement de la file.

Technologies : JAVA , Android, xml, Json, Postgresql, API Rest, Express Framework, Git, Android Studio
Responsabilité  : Backend(Api, modélisation de la Base de donnee) et gestion de la logique de la file d’attente..
